## How I carve

```text
Prompt → Slice → Contract → Test → Deploy → Observe → Carve
```

- 도메인 단위의 작은 수직 기능으로 범위를 고정합니다.
- API·데이터·UI 사이의 계약을 코드와 테스트로 남깁니다.
- 빌드와 배포를 자동화하고, 실패 시 돌아갈 기준점을 먼저 만듭니다.
- 실제 UI, 테스트 결과, 릴리스 기록처럼 다시 확인할 수 있는 증거를 우선합니다.
